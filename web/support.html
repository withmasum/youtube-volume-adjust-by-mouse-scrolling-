Support

If you have questions or need help with the YouTube Volume Control extension, feel free to reach out or report any issues. Below are the ways to contact us and get support.

Contact Information

  • Facebook: facebook.com/masumnotes
  • GitHub: github.com/masumnotes/YouTube-Volume-Scroll
  • Email: support@masumnotes.com

Reporting Issues

If you encounter any bugs or have feature requests, please report them on our GitHub Issues page or send us an email. When reporting an issue, include as much detail as possible (such as your browser version, extension version, and steps to reproduce the problem) to help us address it quickly.

Before submitting a new issue, you may want to check the existing GitHub issues to see if it has already been reported.
